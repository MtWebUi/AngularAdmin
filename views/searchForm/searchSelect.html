• 案例

  • 使用

    <ui-search-select></ui-search-select>

  • 属性

    属性名 默认值 是否必须 描述
    ref $searchSelect 否 类似id, 用于在scope中获取
    name 是
    model 否 转换为ng-model
    url 否 如果要远程加载数据的话
    key-name key 否 如果远程加载数据, 那么拿这个属性作为数据条目的value
    value-name text 否 如果远程加载数据, 那么拿这个属性作为数据条目的label

  • 方法

    方法名 描述
    load(url) 远程加载数据
    setData(data[下拉数据, 如果要分组显示就传json, 如果不分组就传list], isClean[是否清空之前], keyName[数值kye的名称], valueName[数值显示的名称]) 动态设置下拉显示的数据
    change(callback[回调函数]) 监听数据变动事件
    val(v) 类似jquery的val
