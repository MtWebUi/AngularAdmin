• 案例

  • 使用

    <ui-search-input ></ui-search-input>

  • 属性

    属性名 默认值 是否必须 描述
    ref $searchInput 否 类似id, 用于在scope中获取
    type text 是 输入框类型
    name 是 提交到后台的数据
    model 否 转换成ng-model

  • 方法

    方法名 描述
    val(v) 类似jquery的val作用
    reset() 清空数据
    disabled(boolean) 是否禁止
    attr(attrName, attrValue) 类似jquery的prop和attr
