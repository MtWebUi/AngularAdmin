• 案例

  • 使用

    <ui-search-date-range></ui-search-date-range>

  • 属性

    属性名 默认值 是否必须 描述
    ref $searchDateRange 否 类似id, 用于在scope中获取
    form-name 是 开始时间的name
    to-name 是 结束时间的name
    from-model 否 开始时间的ng-model
    to-model 否 结束时间的ng-model
    mode datetime 否 date为日期, datetime为日期+时间
    limit 否 如果想将时间区域的范围设置只能某一段间隔内的话, 设置该值(单位是天)

  • 方法

    方法名 描述
    val(startValue, endValue) 和jquery的val类似
    change(callback[回调函数]) 监听数据变动事件
