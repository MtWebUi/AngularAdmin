• 案例

  • 使用

    <ui-search-date></ui-search-date>

  • 属性

    属性名 默认值 是否必须 描述
    ref $searchDate 否 类似id, 用于在scope中获取
    name 是
    model 否 转换为ng-model
    mode datetime 否 date为日期,time为事件,datetime为日期+时间

  • 方法

    方法名 描述
    val(v) 和jquery的val类似
    change(callback[回调函数]) 监听数据变动事件
    reset() 清空数据
