UIForm

  • 使用

    <ui-form>
    </ui-form>

  • 属性

    属性名 默认值 是否必须 描述
    ref $form 否 类似id,用于scope
    url 否 适用于编辑场景, 远程获取表单数据
    action 否 如果设置了该值, 那么将会自动提交
    form-name 否 表单的验证规则, 远程对象

  • 方法

    方法名 描述
    setData(data[表单数据]) 设置表单数据
    setUrl(url) 远程获取表单数据
    layout(column[一行几列]) 自动布局
    submit(callback[回调]) 提交数据触发

  • 事件

    事件名 参数 触发
    uiform.reset (evt) 当点击清空按钮触发
    uiform.doSubmit (evt, formData) 当点击提交按钮触发
    uiform.rules (evt, rule) 当获取到验证规则触发, 用于动态修改验证规则
